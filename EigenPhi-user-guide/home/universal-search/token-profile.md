# Token Profile

**Token Summary**: shows token address, symbol, decimals, and if it is a malicious token.      &#x20;

**Performance of MEV Types:** shows the token's related MEV performance.      &#x20;

**Liquidity Pools:** This section displays the liquidity pools involved with the selected token, sorted by remaining volume. Clicking on the address of a liquidity pool will take you to the Liquidity Pool Profile.           &#x20;

<figure><img src="../../.gitbook/assets/image (339).png" alt=""><figcaption></figcaption></figure>

**Latest MEV:** Like the MEV Live-stream data on the homepage, this section shows the latest MEV transactions involved with the selected token.     &#x20;

<figure><img src="../../.gitbook/assets/image (349).png" alt=""><figcaption></figcaption></figure>

**MEV Proft Leaderboard:** This section shows the MEV transactions with the highest profit/ROI (click to sort by profit/ROI) involved with this token, and you can see their types, contracts, profit, ROI, and time. Click on a specific transaction to get more details.
