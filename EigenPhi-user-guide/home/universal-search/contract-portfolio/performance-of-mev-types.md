# Performance of MEV Types

In this section, data and charts can be sorted by profit or volume.     &#x20;

The chart displays the trading profit data of this contract. The left vertical axis shows the profit amount, and the right vertical axis measures the MEV frequency of MEV events.&#x20;

You can also check out the breakdown of profit/volume for each day. The different colors stand for different MEV types. By hovering on the stacked column, you can see the exact amount.
