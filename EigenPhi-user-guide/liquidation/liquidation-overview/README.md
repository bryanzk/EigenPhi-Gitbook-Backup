# Liquidation Overview

**Liquidation  Overview** provides the big picture of the liquidation market, which is correlated with the fluctuation in the market. All the data can be filtered by 24H/7D/30D.

This page has six sections, the first of which is **Summary,** showing a series of data here:

![](<../../.gitbook/assets/image (270).png>)

The **Trend** chart gives you the daily trend of the selected period. Here we can see that there were 63 liquidations that happened on Aug 14, with the total amount as almost $24K.&#x20;

![](<../../.gitbook/assets/image (238).png>)

The logic of **Profit Distribution** here is the same as the [MEV Transaction Profit Distribution](../../home/mev-transaction-profit-distribution.md) on the homepage, except it focuses on Liquidation.

The figure below shows that there are 7 liquidations with profits between $100 and $1000 that happened during the selected period, and the average profit is $291.81.

![](<../../.gitbook/assets/image (283).png>)

Next is the **Latest Liquidation** list, showing all the liquidations in chronicle order. Other than revenue, liquidation amount, and [liquidation LTV](https://eigenphi-1.gitbook.io/classroom/glossary/liquidation/ltv), the other 4 columns in the list can be examined in detail.

* **Tx** hash takes you to the [liquidation transaction detail](../liquidation-transaction.md).&#x20;
* The **Liquidator** address will lead you to the [liquidator detail](../liquidator-details.md).
* The **Liquidation Asset** column's token symbols show you the [detailed token profile](../../home/universal-search/token-profile.md).
* The **Protocol** column's data takes you to the [Liquidation Protocol Overview](../liquidation-protocol-overview.md) page.&#x20;

![](<../../.gitbook/assets/image (259).png>)

We will examine the Liquidation Leaderboard in detail later. Let's take a look at the **Protocols** section. It lists the liquidation protocols ranked by liquidation amount. Other related data are here, too, for researchers to have a first impression of the protocol. Click the protocol name to find out the[ liquidation details of this protocol](../liquidation-protocol-overview.md).

![](<../../.gitbook/assets/image (67).png>)
