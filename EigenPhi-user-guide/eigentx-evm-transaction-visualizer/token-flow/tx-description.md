# Tx Description

Another section is  "Transaction Description" which is a general description of what happened during this Tx.

It includes the following information: &#x20;

* Exact time the Tx happened on&#x20;
* MEV type
* Tokens and trading venues involved
* &#x20;Profit and Loss
* Token Price
