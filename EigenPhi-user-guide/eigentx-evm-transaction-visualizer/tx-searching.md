# Tx Searching

When users type in the Tx hash they'd like to search themselves, a more detailed page will be prompted as following. There are four tabs -- "Token Flow", "Block" , "Tx Involving From" and "Tx involving To". Each of them contains different information that will enhance your experience of interacting with MEV.&#x20;
