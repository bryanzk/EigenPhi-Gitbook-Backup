# Tx involving From

If you are interested in specific addresses of the Tx, you can explore more on the last 2 tabs. \
"Tx involving From" means all the transactions that involve the same "From" address from this Tx. &#x20;

<figure><img src="../.gitbook/assets/image (310).png" alt=""><figcaption></figcaption></figure>

It's important to note that when users utilize this function, all transactions related to the "From" address of the initial transaction search will be displayed in the list, regardless of whether the address is listed as the "From" or "To" address in other transactions.
