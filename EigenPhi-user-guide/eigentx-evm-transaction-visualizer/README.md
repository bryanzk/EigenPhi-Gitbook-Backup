# EigenTx EVM Transaction Visualizer

EigenTx EVM Transaction Visualizer is a powerful tool for tracking and analyzing complex txs on EVM compatible chains. In addition to what [Public Version of EigenTx ](https://tx.eigenphi.io/analyseTransaction)offers, which is a simpler and quicker tool for users to analyze transactions, EigenTx EVM Transaction Visualizer has incorporated multiple new features that enable users to have a more comprehensive and customized experience of understanding transactions.

Followings are the EVM compatible chains that EigenTx supports.&#x20;

* Ethereum
* BSC
* Avalanche
* Polygon
* Arbitrum
* Optimism
* Cronos
* Fantom
