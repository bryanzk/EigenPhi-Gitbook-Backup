# Layout & More Settings

Right next to the searching bar, users can also customize how they‘d like to see the "Token Flow" chart. There are 4 Layouts supported, Top to Bottom (TB), Bottom to Top(BT), Left to Right (LR), and Right to Left (RL). If there is no preferred layout specified by users, EigenTx will automatically choose the best layout by default.&#x20;

<figure><img src="../../.gitbook/assets/image (321).png" alt=""><figcaption></figcaption></figure>

There is also a button called "More Settings" which gives users more options of how they like token flow chart to be graphed.&#x20;

* Show Net Asset Flow: Simplify the steps when same token loops between two addresses, show net transfer only
* Hide Inter-mediate Asset Transfer: Skip the middle node when transfer the same token with same amount
* Allow Different Transfer Amount: Skip the middle node when transfer the same token
* Disable Reverse Debt Token Transfer: Show the original debt token transfer
