# Search Pane

In the search plane, there are multiple options you can select to adjust the outcome. Chain Selection allows you to choose the chain that the transaction you are looking for is on. You also have several options that can help you simplify the graph output. The options include eliminating the transfer with the same token and the same/opposite from to, eliminating the transfer with the same token and the same middle node. Besides, in reality, the debt token is minted directly and then transferred to the user, but we reverse the direction so that it can form a cycle on the graph. You can choose to avoid this operation. Furthermore, you can also use the layout option to change the graph layout.

### Option

* Choose the chain that the transaction you are looking for is on.
* You can use several options below to simplify the graph.
