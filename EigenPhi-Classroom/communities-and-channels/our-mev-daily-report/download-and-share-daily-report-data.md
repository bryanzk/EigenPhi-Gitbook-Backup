# Download And Share Daily Report Data

To download the data, go to the header of the report and click the Download icon.

![](<../../.gitbook/assets/image (6).png>)

You can choose  download the current report's data in the format of either PDF or CSV. Interested in more historical data? Wanna find out the trends over a longer period? Download HIstory Data option will give you a CSV file containing data starting from Jan 1st, 2022.&#x20;

![](<../../.gitbook/assets/image (24).png>)

Of course, you can share the report via Twitter, Telegram, and Reddit.&#x20;

That’s it for the Ethereum MEV daily reports. We have generated them daily since the beginning of 2022, which you can examine in the [Overview section](overview-section-of-daily-report.md).&#x20;
